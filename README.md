# vue-demo vue的测试程序。
项目配置好了axios和element ui，封装了axios的调用和方法调用。

0.环境配置
首先需要安装node.js,然后安装vue-cli   cnpm install -g @vue/cli 
1.vue create <Project Name> //文件名 不支持驼峰（含大写字母）
创建项目
